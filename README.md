# People Analytics: Attrition Modeling and NLP Analysis

This project demonstrates how structured HR data and unstructured exit interview text can be combined to generate workforce insights.

## Objectives

- Identify workforce segments associated with high attrition
- Build predictive models for employee departures
- Apply NLP to exit interview comments
- Extract actionable insights for workforce planning

## Methods

- Exploratory data analysis
- Feature engineering
- Logistic regression modeling
- Natural language processing of exit interview text

## Dataset

This project uses a simulated employee dataset for demonstration purposes.
